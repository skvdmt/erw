# Простая обертка ошибок

## Переводы
[En](./README.md)

## Описание
Библиотека предназначенная для оборачивания ошибки для ее дальнейшей обработки.
Имеет возможность формировать и передавать код статуса ошибки, сообщение предназначенное клиенту и детализацию ошибки для сохранения в журнале.

## Установка
```sh
go get github.com/skvdmt/erw
```
