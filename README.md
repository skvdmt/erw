# Simple error wrapper

### Translations
[Русский](./README_ru.md)

### Description
A library designed to wrap an error for further processing.
It has the ability to generate and transmit an error status code, a message intended for the client, and error details for saving in the log.

### Install
```sh
go get github.com/skvdmt/erw
```
